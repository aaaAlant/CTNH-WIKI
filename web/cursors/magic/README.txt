Magic cursor theme assets go here.

Maintain `manifest.json` in this folder and list the file names you want to use.

Example:
[
  "bloodmagic3-sacrificialknife.png",
  "botania-wandoftheforest.png"
]

The hotspot is currently configured as 8 8 in web/index.html.
When the magic theme is activated, one available candidate is chosen at random.
